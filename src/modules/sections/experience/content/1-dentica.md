title: Full-Stack Developer
---

- Created an application for scheduling visits with a specific dentist using React, Electron.js, TailwindCSS, and Firebase which includes:
  - Developed a system for the dentist to easily create and manage patient records, including their appointment history and procedures performed
  - Implemented a panel for the dentist to use for scheduling and managing appointments for  patients. This included features such as the ability to view their schedule, add or remove appointments, and make any necessary changes to existing appointments.
  - Created a system for the dentist to accurately document procedures performed on patients during appointments, as well as plan any further work that may be needed
  - Integrated an SMS API (serwersms.pl) that sends notifications to patients via text message for scheduled, cancelled, or rescheduled visits, as well as a reminder the day before the appointment
  - Worked closely with the dentist on creation and improvement of the scheduling app
